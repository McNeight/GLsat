Case Study: Two Translations

 Experiment

In this experiment we examine the following code segment:

    glTranslatef(x1,y1,z1);
    glutSolidTeapot(1.0);
    glTranslatef(x2,y2,z2);
    glutSolidSphere(1.0,8,8);

Since the effect of a transformation command applies to all subsequent transformation and rendering commands (as seen in the previous experient), the position of the teapot is affected by the first glTranslatef() command only, while the position of the sphere is determined by both glTranslatef() commands.

Click here to run an example. Initially the code looks like
    glTranslatef(0,0,0);
    glutSolidTeapot(1.0);
    glTranslatef(3,0,0);
    glutSolidSphere(1.0,8,8);
so the teapot appears at the origin and the sphere is located 3 units in the positive x direction. Now try this:

- Change x1, y1, z1 of the first glTranslatef() to 2, 0, 4, respectively. You see that both the teapot and the sphere are relocated, since the change affects both objects.

- Now change x2, y2, z2 of the second glTranslatef() to any values you like. This affects only the sphere and changes its position relative to the teapot.
