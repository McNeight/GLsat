Case Study: Interchanging Translation and Rotation

Experiment

In this experiment we compare
    glTranslatef(x1,y1,z1);
    glRotatef(angle,x2,y2,z2);
    glutSolidTeapot(1.0);
and
    glRotatef(angle,x2,y2,z2);
    glTranslatef(x1,y1,z1);
    glutSolidTeapot(1.0);
The difference between the two code segments is that glTranslatef() and glRotatef() appear in different orders.

Click here to run the example. The program contains both code segments but one of them is commented out. You can replace it with the other segment by clicking a suitable radio button.

Initially the program uses
    glTranslatef(0,0,4);
    glRotate(60,0,1,0);
    glutSolidTeapot(1.0);
and the teapot appears somewhere on the z axis. Now click the button "Rotate then Translate" followed by "Execute", to run the program using
    glRotate(60,0,1,0);
    glTranslatef(0,0,4);
    glutSolidTeapot(1.0);
Observe that now the teapot appeares in a different location that we hope you have guessed correctly.

Now it is your turn to change the parameters to any values you like, switch between the two code segments, and test your knowledge on the effect of multiple transformations. If you have any questions please go back to "Effects of Multiple Transformations".
