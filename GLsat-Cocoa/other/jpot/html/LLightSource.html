Light Source Position

Idea:

The routines GLfloat light_position[] = {x, y, z, w} and glLightfv(GL_FLOAT0, GL_POSITION, light_position) define the position of the light in the scene. If w = 0 the light source is considered an infinite one (i.e. directional), the ray of light can be considered parallel by the time they reach the object (like the sun), and the x, y, and z values specify the direction. If w = 1 the light source is positional (like the desk lamp), the x, y, and z values specify the exact location of the light source in the scene.
