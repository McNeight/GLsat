Spotlights and Multiple Lights

Spotlights:

A positional light source can act as a spotlight by restricting its shape to that of a cone as in the picture below:

To create a conical spotlight, we must first decide upon its position as well as the angle at its apex. Position the spotlight using commands exactly as for a regular positional light source (remember with a positional light, always w = 1). Use the GL_SPOT_CUTOFF parameter to specify a value equal to one half the angle at the apex of the cone. By default the GL_SPOT_ CUTOFF parameter is set to 180.0 degrees so effectively the spotlight feature is disabled. The default direction is (0.0, 0.0, -1.0), that is the negative Z direction. The GL_SPOT_EXPONENT parameter determines how focused the spotlight is, in other words, how rapidly its intensity diminishes from the central ray through the cone to its sides. Attenuation factors are set exactly as for a regular light source.
