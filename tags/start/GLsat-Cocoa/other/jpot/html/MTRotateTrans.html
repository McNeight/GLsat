Case Study: Translation followed by Rotation

 Experiment

In this experiment we examine the following code segment:

    glTranslatef(x1,y1,z1);
    glRotatef(angle,x2,y2,z2);
    glutSolidTeapot(1.0);
Click here to run the example. Initially the code looks like
    glTranslatef(0,0,0);
    glutSolidTeapot(1.0);
    glTranslatef(3,0,0);
    glutSolidSphere(1.0,8,8);
so the teapot appears at the origin and the spout is facing the positive x direction. Now try this:

- Change x1, y1, z1 of the glTranslatef() to 2, 0, 4, respectively. Did the teapot appear at the location you expected?

- Now change angle, x2, y2, z2 of the glRotatef() to 60, 0, 0, 1, respectively, or any values you choose. Again, did the teapot appear at the location and in the orientation you expected? If not, you may want to go back to "Effects of Multiple Transformations".
