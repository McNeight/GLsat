Drawing Quads

 Ideal

Quad is a geometric primitive in OpenGL that is specified by sets of 4 vertices. If GL_QUADS mode is used in the glBegin() command, a succession of quadrilaterals is drawn using vertices v0, v1, v2, v3 then v4, v5, v6, v7 and so on. If number of vertices is not a multiple of 4, the last one to three vertices will be ignored.

If GL_QUAD_STRIP is used in the glBegin() command, a succession of quadrilaterials is drawn using vertices v0, v1, v3, v2, and then v2, v3, v5, v6, and so on. The final vertex is ignored if the number of vertices is odd.
