Moving Light Source With Viewpoint

Description

To create a light that moves along with the viewpoint you need to set the light position before the viewing transformation. Then the viewing transformation affects both the light and the viewpoint in the same manner.

The example shows the torus of the experiment with a moving light source, except that now the view point and the light source are both located at a location that travels around the torus (we dont see the cube any more because the eye is located at it). The left mouse button starts the animation. Observe how the visible portion of the torus is always lit.
