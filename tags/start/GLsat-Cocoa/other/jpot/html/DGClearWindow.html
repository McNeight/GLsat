Window Clearing

Idea:

We had a brief look at changing the background color in the Background Color lesson. We try to understand the theory here.

In a computer, the frame holding the picture is usually filled with the last picture drawn, so typically one need's to clear it to some background color before starting to draw a new scene. As an example, these lines of code clear an RGBA mode window to black:

glClearColor(0.0, 0.0, 0.0, 0.0);

glClear(GL_COLOR_BUFFER_BIT);

The first command sets the clearing color to black, and the next one clears the entire window to the current clearing color. The parameters to glClear() indicates which buffers are to be cleared. In this case, the program clears only the color buffer. Typically you set the clearing color once, early in your application. At this point, it would not be a bad idea to review some of the examples in the first chapter to see how the above lines of code are positioned with regards to the overall structure of the program. Also other buffers might be encountered. Listed below are all OpenGL buffers :

Buffer Name

Color

GL_COLOR_BUFFER_BIT

Depth

GL_DEPTH_BUFFER_BIT

Accumulation

GL_ACCUM_BUFFER_BIT

Stencil

GL_STENCIL_BUFFER_BIT

Before issuing a command to clear multiple buffers, you have to set the values to which each buffer is to be cleared. Further you can clear multiple buffers by bitwise ORing them in the glClear() command:

glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT);
