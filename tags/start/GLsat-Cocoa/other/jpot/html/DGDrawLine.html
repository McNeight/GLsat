Drawing Lines

 Ideal

A line is a geometric primitive in OpenGL that is specified by its two end points, which are, of course, in turn specified by vertices. Following a glBegin(GL_LINES) command we declare successive vertices. Successive pairs of vertices are joined by lines (if the number of vertices is odd the last one is ignored). To control the width of a line, we use the glLineWidth(GLFfloat size) command where size is the desired width.
