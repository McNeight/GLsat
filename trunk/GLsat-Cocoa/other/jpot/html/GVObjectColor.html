Object Color

Idea:

Object color (or foreground color, in the other words) is defined by using the command glColor3f(x, y, z) where x, y and z are floating-point numbers to specify the color components red, green and blue respectively. If x, y, or z is specified outside this range then it is clamped to 0, for values less than 0, and clamped to 1, for values greater than 1.

When an object is drawn, it will be drawn with the currently specified foreground color.
