Background Color

Idea:

The background color of the window is selected by using the command glClearColor(x, y, z) where x, y and z are the color components red, green and blue respectively. Values of x, y, and z are floating-point numbers in the range 0 to 1. If x, y, or z is specified outside this range then it is clamped to 0, for values less than 0, and clamped to 1, for values greater than 1. To actually clear the window to the specified background color, glClear(GL_COLOR_BUFFER_BIT) is called after glClearColor(x, y, z).
